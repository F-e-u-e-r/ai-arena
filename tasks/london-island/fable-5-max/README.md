# Little London in the Sky — 漂浮倫敦島與四季天氣

卡通風格的 3D 漂浮倫敦島：大笨鐘（時鐘會走）、倫敦眼（吊艙永遠直立）、
倫敦塔橋（小船靠近會開橋）、泰晤士河與雙瀑布、紅色雙層巴士與電話亭。
15 秒一輪日夜、四季氣候平滑輪轉：春霧 → 夏日 → 秋楓 → 冬雪。

## 如何啟動

```bash
# 方法一：直接開啟（需可連網載入 Three.js CDN）
open index.html

# 方法二：任一 static server
python3 -m http.server 8000
# → http://localhost:8000/
```

不依賴後端與建置工具；`main.js` 為 classic script，`file://` 直接開啟也能執行。

## Three.js 版本與引入方式

- **版本**：`three@0.149.0`（≥ v134，UMD 版，掛在全域 `THREE`）
- **引入**：`<script defer src="https://cdn.jsdelivr.net/npm/three@0.149.0/build/three.min.js">`
- 刻意選 UMD 而非 ES module：sandbox iframe 與 `file://` 環境下不受
  module CORS 限制，單檔即可執行。
- 無外部紋理與模型 —— 所有貼圖（光暈、水紋、瀑布條紋）皆以 Canvas 2D 程序化生成，
  所有物件皆由基本幾何體（Box / Cylinder / Torus / Capsule / Shape…）組合。

## 日夜循環（DayNightCycle）

- `dayT ∈ [0,1)` 每 **15 秒**走完一輪：`0` 日出 → `0.25` 正午 → `0.5` 日落 → `0.75` 午夜。
- **關鍵影格取樣**：10 個時刻停靠點（stops），每個停靠點描述天頂色、地平色、霧色、
  日光色/強度、月光強度、半球光、星星透明度、**燈光開關值 lamp**、霧密度。
  取樣時找出前後兩個停靠點做線性插值（含 0↔1 環狀回繞），
  所有屬性因此同步且連續漸變 —— 不會出現任何跳變。
- **自動亮燈**：`lamp` 欄位在日落停靠點（t=0.5）開始爬升、黃昏（t=0.575）到 1，
  黎明前（t=0.925→0.962）降回 0。大笨鐘鐘面與窗、倫敦眼霓虹、塔橋燈串、
  街燈、電話亭全部共用這條曲線（材質自發光強度 + 少量 PointLight）。
- **太陽與月亮**：`sunDir = normalize(cos θ, sin θ·0.66, 0.5)`，θ = dayT·2π；
  月亮取反方向。太陽 y 分量壓到 0.66 讓正午仍是斜射光，陰影不會消失。
  太陽/月亮的網格 + 光暈 sprite 跟著方向移動，低於地平線自動隱藏。
- **大笨鐘時鐘**：場景時刻 `hours = dayT·24 + 6`，四面鐘的時針/分針每幀依此旋轉。

## 四季氣候（SeasonSystem）

- 每季 **20 秒**，`yearT` 連續累進。每季前 75% 為純季節，
  **最後 25% 與下一季做 smoothstep 混合**（跨季過渡窗）。
- 季節參數表：草地/樹冠/河水顏色、霧倍率與霧色調、日光倍率與色調、
  春霧量、楓葉生成率、雪花生成率、積雪量（roofs/道路白化）、花量、風力。
- 另外維護一份「實際套用值」`cur`，以指數平滑（`1-e^(-1.6·dt)`）趨近目標 ——
  **手動切換季節**（按鈕 / 快捷鍵）也因此獲得約 2 秒的柔順過渡，不會瞬間跳變。
- 季節特徵：
  - **春**：霧密度 ×2.7 + 低空霧 sprite 漂移（濕潤水氣感）、櫻粉樹冠、小花盛開
  - **夏**：日光 ×1.16、霧幾乎歸零、色彩最飽滿
  - **秋**：楓葉每秒 20 片生成，樹冠轉橘紅、暖霧金光
  - **冬**：雪花每秒 48 片，草地/屋頂/道路漸白、冷藍日光、河水轉冰藍

## 粒子系統與效能策略

- **InstancedMesh**：楓葉（340 實例，ShapeGeometry 楓葉剪影 + 每片固定秋色
  instanceColor）與雪花（640 實例，六邊形）各只佔 **1 個 draw call**。
- **物件池（free list）**：固定大小的狀態陣列 + 空閒索引堆疊。
  粒子死亡 → 索引 push 回池；生成 → pop 重設 —— 運行期**零記憶體配置、零 GC 壓力**。
- **生成速率累加器**：`acc += rate·dt`，每滿 1 生成一顆；速率直接取自季節插值，
  跨季時粒子量自然升降（秋→冬時楓葉漸稀、雪花漸密）。
- **楓葉物理**：等速下落 + `cos(相位)` 左右擺動 + 雙軸自旋 + 全場緩慢變向的微風；
  落在島面上會停留 2.6 秒漸隱回收，落出島外墜至 y<-30 回收。
- 其他效能措施：
  - 所有 update 共用模組層級暫存物件（`_c1`/`_v1`/`_dummy`），主迴圈不 new 物件
  - `deltaTime` 上限 0.1s（分頁喚醒不會物理暴衝）；場景時間採 dt 累加制
  - 貼圖滾動（UV offset）做河水流動與瀑布，免 shader、免頂點更新
  - HUD 文字每 0.5s 才更新一次；按鈕狀態變化偵測後才碰 DOM
  - `pixelRatio` 上限 2；陰影僅太陽一盞 2048 貼圖；PointLight 總數 4

## 相機旋轉控制（自行實作）

- **Pointer Events** 統一滑鼠與觸控：拖曳改變方位角/仰角（帶夾角限制），
  放開後角速度以 `e^(-3.2·dt)` 指數衰減（慣性感）。
- 滾輪與雙指捏合縮放（36–100 clamp）。
- 閒置 4 秒後緩慢自動環繞（gallery 預覽時畫面自帶生命力），一觸碰立即讓位。
- 拖曳全程不打斷 `requestAnimationFrame` 主迴圈 —— 動畫持續運行。
- 小視窗自適應：viewport 高度 < 560px 時自動拉遠鏡頭讓整島入鏡
  （使用者手動縮放後即不再干預）。

## 操作

| 操作 | 效果 |
| --- | --- |
| 拖曳 / 單指滑動 | 旋轉視角 |
| 滾輪 / 雙指捏合 | 縮放 |
| `1` `2` `3` `4` 或底部按鈕 | 手動切換 春/夏/秋/冬 |
| `A` 或 A 按鈕 | 恢復季節自動輪轉 |
| `M` 或 🔊 按鈕 | 環境音效開關（風聲隨季節、日落時大笨鐘敲三響、正午一響） |
