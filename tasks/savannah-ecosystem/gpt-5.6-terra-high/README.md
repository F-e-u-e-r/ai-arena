# Serengeti Pulse

直接開啟 `index.html` 或以任意 static server 執行。作品完全使用原生 Canvas 2D API，適合將大量小型粒子、視差地景與程式化動物保留在一個低負擔的繪圖管線中。

畫面有 6 隻動物：兩隻獅子會群體慢行、隨機休息；兩隻大象慢速移動，雨時更慢；兩隻豹獨行，且會隨機垂直攀樹。牠們持續由左循環至右，切換狀態不會重設位置。濕／乾季每 60 秒自動輪轉，也可手動控制，會改變景色與動物步速。

ParticlePool 在啟動時配置 380 個可重用物件；天氣切換會回收舊粒子。雨、豪雨、雪、雷暴、龍捲風都從同一池借用粒子。雷暴會在離所有動物水平安全距離最大的候選點落雷；龍捲風則每幀尋找最近動物後慢速追近。主迴圈使用 rAF 的 delta time，版面在 320×200 時控制鈕會壓縮而不遮蔽場景。
