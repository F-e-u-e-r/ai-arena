# Savannah / Living Weather

一個只使用原生 Canvas 2D API、無框架、無外部函式庫與無後端的非洲大草原生態系統。畫面包含兩隻獅子、兩隻大象和兩隻豹，以及可切換天氣、乾濕季、視差背景、動物狀態和粒子特效。

## 啟動

可直接開啟 `index.html`，或在此資料夾啟動靜態伺服器：

```bash
python3 -m http.server 8000
```

再開啟 `http://localhost:8000/`。

## Canvas 與架構

- Canvas 內部採固定 960×600 世界座標，再依容器尺寸等比映射，確保 320×200（16:10）預覽仍有穩定構圖。所有動物、背景、天氣和 HUD 外的場景繪製均透過 Canvas 2D API。
- `Ecosystem` 管理時間、日夜、季節、天氣與六隻角色；`Lion`、`Elephant`、`Leopard` 都繼承 `Animal`，各自有不同速度、群聚/休息、喝水、抬鼻、巡邏、潛行和攀爬狀態。
- 動物持續由左往右移動，越過右側後回到左側，不因換天氣或季節重設位置。濕季提高活動速度；乾季讓獅群和象群更趨向慢速群聚；雨天降低獅子速度。
- `SeasonManager` 以 60 秒自動輪轉乾季/濕季，並在 3.8 秒內平滑混合天空、遠山、草地、地面、水面和整體亮度；按季節按鈕或 `S` 可立即開始切換。

## 粒子池與天氣

`ParticlePool` 啟動時預先配置 560 個粒子，分成雨、雪、塵土和夜間螢火蟲。天氣切換先呼叫 `clear()` 將所有 active 粒子放回 free list，之後只用 `acquire()` 重用它們，動畫中不會持續 new / dispose 粒子。雨天有積水波紋，龍捲風會緩慢朝最近動物移動並捲起塵土。

雷暴每次從 42 個天空候選欄位中計算與所有動物的最小水平距離，挑選最空的欄位；閃電的折線寬度小於這個安全半徑，且閃光持續期間每幀重新計算安全走廊，因此不會擊中動物。畫面會短暫標示 `SAFE STRIKE`。

主迴圈使用 `requestAnimationFrame` 和基於時間戳的 `deltaTime`，並將單幀 delta 限制在 50ms，避免分頁切換後角色或粒子跳躍。按鈕下排在窄螢幕改用短標籤，天氣六選一和季節按鈕在 320×200 預覽仍可點擊。
