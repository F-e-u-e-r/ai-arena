# 非洲大草原生態系統（Canvas 2D）

由 Claude Fable 5（effort: max）以原生 HTML + CSS + ES6 JavaScript 生成，
不使用任何框架或外部函式庫，渲染完全基於 Canvas 2D API。

## 如何啟動

```bash
# 直接開啟
open index.html

# 或使用簡單 static server
python3 -m http.server 8000
# → http://localhost:8000/index.html
```

無後端、無建置步驟。所有視覺元素（動物、地景、粒子）皆以程式繪製，未使用外部 Sprite 或紋理。

## 操作

| 操作 | 說明 |
| --- | --- |
| 底部按鈕 | 切換 6 種天氣（晴 / 雨 / 豪雨 / 雪 / 雷暴 / 龍捲風）與乾濕季 |
| `1`–`6` | 天氣快捷鍵 |
| `S` | 手動切換季節（也會每 60 秒自動輪轉，季節鈕下方有進度條） |
| `D` 或 ⓘ | 顯示 FPS / 粒子數 debug 資訊 |

## 為何選擇 Canvas

場景中同時存在數百至上千個獨立粒子（雨、雪、碎屑）、8 隻各自帶骨架動畫的
動物與多層視差背景。以 DOM/SVG 表達會造成大量節點與 reflow；Canvas 2D 提供
單一繪圖表面 + 立即模式渲染，可以逐幀完全掌控繪製順序（深度排序）、批次化
同型別粒子，並把每幀成本壓在可預測範圍內。

## 系統架構

```
App                 rAF + deltaTime 主迴圈、resize、HUD、深度排序
├─ SeasonManager    乾/濕季：60s 自動輪轉、3s 色彩過渡、行為倍率
├─ WeatherManager   天氣狀態機、閃電排程、積水/積雪、Tornado 實體
├─ ParticleSystem   分型別 Object Pool（雨/雪/水花/碎屑/塵土）
├─ Lion ×3          獅群錨點跟隨、巡邏/趴下休息
├─ Elephant ×3      象群錨點跟隨、水塘喝水、舉鼻（含幼象）
├─ Leopard ×2       疾行/潛行交替、攀爬金合歡樹（三階段動畫）
└─ Scenery          天空/遠山/金合歡/水塘/草叢/前景（視差多層）
```

所有動物走在「未包裹座標」`xU` 上持續向右，顯示座標
`displayX = mod(xU, 畫面寬 + 緩衝) - 緩衝`，因此右側離開後從左側無縫再進場，
群體錨點跨越邊界也不會斷裂，切換天氣或季節不會重置任何動物位置。

## 粒子管理與效能策略（Object Pooling）

- 每種粒子有**獨立固定容量的池**（雨 1500 / 雪 520 / 水花 360 / 碎屑 220 / 塵土 260）。
  粒子物件在啟動時一次配置完畢，執行期**零配置**。
- 池採「活躍前綴」佈局：`spawn()` 取 `items[count++]`；回收用 swap-with-last
  （把死亡粒子與最後一個活躍粒子交換後 `count--`），O(1) 且無洞。
- 池滿時直接放棄生成（丟幀粒子比掉 FPS 好）。
- **批次繪製**：同型別粒子共用一次 `beginPath/stroke(fill)`，雨滴整池只有一次
  stroke 呼叫，避免每粒子切換樣式。
- 效能守門：FPS 持續低於 42 時自動把生成倍率 ×0.75（下限 0.4）；
  `prefers-reduced-motion` 時倍率上限 0.5 並停用畫面震動。
- `deltaTime` 上限 50ms：分頁切回時不會產生粒子噴發或動物瞬移。

## 雷暴安全區演算法

閃電**永遠不會擊中動物**，做法（`WeatherManager.strike`）：

1. 落雷當下（非預先排程）把每隻動物投影到 x 軸，展開成區間
   `[displayX − 半身長×2.5 − 邊距, displayX + 半身長×2.5 + 邊距]`。
2. 區間排序後合併，對畫面內側 `[6%W, 94%W]` 取補集 → 得到安全區段集合。
3. 依區段長度加權隨機取一點作為落雷 x；若沒有足夠寬的安全區段，
   本次退化為只有天空閃光的「遠雷」。

因為計算發生在落雷的同一幀，動物移動不會產生時間差誤擊。

## 龍捲風追逐

龍捲風鎖定 `displayX` 最近的動物，以 `14 × 畫面縮放` px/s 的上限速度逼近 ——
刻意低於最慢的大象步速（20 × 縮放），因此構成持續威脅但永遠追不上；
接近的動物會進入恐慌狀態（最高 2.1× 加速 + 😨）。漏斗由 9 節橢圓組成，
各節以「越高越慢」的速度跟隨軸心，自然拖出鞭甩感；底部持續把草屑
（debris 池）以螺旋路徑捲上天。切換天氣時漏斗上升消散，殘餘粒子自然回收。

## 動物習性與季節邏輯

| 動物 | 習性實作 |
| --- | --- |
| 獅子 | 3 隻（1 公 2 母）共享「獅群錨點」：落後快走、超前放慢；偶爾趴下休息（💤），乾季休息頻率 ×1.5 |
| 大象 | 3 隻（含幼象）錨點跟隨、步速最慢；行經水塘會停下喝水（💧，鼻尖伸向水面 + 漣漪），偶爾舉鼻（🎺） |
| 花豹 | 單獨行動，疾行（88px/s）與低姿態潛行（🐾）交替；經過大金合歡樹有機率演出「上樹 → 趴枝（尾巴垂晃）→ 下樹」三階段動畫 |

季節影響（60 秒自動輪轉，3 秒平滑過渡）：

- **視覺**：草地由金黃（乾）↔ 翠綠（濕）、天空/遠山/水塘色調與水面大小同步漸變。
- **行為**：濕季速度 ×1.18、動作頻率 ×1.3；乾季群聚間距 ×0.72、更常休息。
- **天氣可用性**：降雪視為乾季（旱冬）限定 —— 濕季時雪按鈕停用；
  下雪中進入濕季會自動轉為降雨並顯示提示。

天氣同時影響行為：豪雨 ×0.8、雪 ×0.62 減速；雷擊會驚動附近動物短暫加速。

## Responsive Embed（320×200 驗證）

- 控制列在小尺寸自動收斂成純 icon（CSS media query ≤520×340），
  8 顆按鈕總寬 <300px，不重疊、不溢出。
- 標題 4.5 秒後淡化至 38% 透明度；debug 預設隱藏，皆不遮擋主場景。
- 所有尺寸（動物、粒子速度、地景）以 `su = clamp(H/520, …)` 統一縮放。

## 長時間運行

- 執行期無新物件配置（粒子池預先配置、env/色盤陣列每幀重用）。
- 無 `setInterval`；唯一計時器是 toast 淡出（會 clearTimeout）。
- 池上限硬性封頂記憶體；30 分鐘以上連續切換天氣不會累積任何陣列。
