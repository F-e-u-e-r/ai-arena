# Wild Rhythm — Savannah Ecosystem

以原生 Canvas 2D 繪製的動態非洲大草原。沒有框架、函式庫、圖片或後端依賴；動物造型、視差環境、粒子及天候效果全部由 Canvas path 即時繪製。

## 啟動與技術選擇

直接開啟 `index.html`，或以任意 static server 提供資料夾。Canvas 2D 很適合這題：每隻動物可用少量 path 呈現有辨識度的輪廓，粒子與多層背景也能在單一 draw call 流程中依深度排序，並能精確控制固定物件池的生命週期。

## 生態與狀態機

- 3 隻獅子有巡邏、休息與群體跟隨狀態；3 隻象低速結隊，會喝水或抬鼻；3 隻豹高速獨行，接近金合歡樹時可能進入攀爬狀態。
- 動物由右側離場後只重設 `x` 到左側，天氣/季節切換不改動任何動物位置。濕季提高活躍與移動速度，乾季則增加休息、喝水與靠近群體中心的傾向。
- `SeasonManager` 每 60 秒切換乾/濕季，用 `seasonMix` 緩慢插值天空、草地、亮度與植被；按鈕可立即指定下一季但仍保留視覺過渡。

## 天氣、粒子與安全

`ParticlePool` 啟動時建立 520 個可重用 plain object。`WeatherManager` 只 acquire/release 這批物件；換天氣時全部回收到 free list，不產生短命陣列或 DOM。雨、大雨、雪、雷暴與龍捲風碎屑共用同一池，離界即重設。

雷暴每次產生閃電前，`chooseSafeStrike()` 會即時掃描所有動物的畫面 x 座標，以動物尺寸加安全半徑建立排除區；最多抽樣 36 次並選擇與最近動物距離最大的候選點，若沒有候選點通過安全距離就取消該次落雷，因此 bolt 的地面端不會落在動物碰撞範圍。龍捲風每幀尋找最近動物，沿 x 軸以低於動物常態速度的上限追逐，形成持續但追不上的壓力。

主迴圈使用 `requestAnimationFrame` 與秒制 `deltaTime`（上限 50ms）。Canvas 依裝置像素比縮放、在 resize 時維持 16:10 友善構圖；320×200 下控制列縮為六個天氣 icon 與季節按鈕。
